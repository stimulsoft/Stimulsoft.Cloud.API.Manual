# Items

The basis of the **Stimulsoft Cloud** functionality is operations on items. The item is an object that can be visually observed in the object tree on the left side of the interface of the client application. All items are inherited from the root abstract class **StiItem** and, there is one of the following classes depending on the functionality provided:


* **StiFolderItem** - provides a hierarchical tree structure elements;

* **StiReportTemplateItem** - report template for building;
