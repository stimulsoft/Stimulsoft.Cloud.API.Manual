# Share

Some elements of the navigator list, can have access from the outside, declared in one of the three levels ShareLevel (Private and Public). You can also set an expiration date of public access. Manually these parameters can be set on the following form:


![](../../../../images/topics/Stimulsoft_Cloud.REST_API.Items.Share_1.png)

Software functionality is described using the command Items Share.


**Name**

**Description**

**GET Info**

Getting information about access parameters of specified item in the cloud of the logged-in user

**PUT Edit**

Changing information about access parameters of specified item in the cloud of the logged-in user.

**DELETE Reset**

Resetting access parameters of specified item in the cloud of the logged-in user to default. These are ShareLevel, established in Private (no public access), ShareMode set in Download, as well as the absence of ShareExpires (period of validity of link of public access).
