# Users

To access information about users of the **Stimulsoft Server**, there is a specialized user class called **StiUserConnection** that automatically handles much of the functionality required for user account management. With this class, you will be able to add user account functionality in your app. In order to use the class **StiUserConnection** you should create an instance of **StiServerConnection** and call one of its methods (**StiServerConnection****.Accounts.****Users**). Access to information about the user provides the **StiUser** class. The user name or the key can obtain an instance of this class. Use for this methods **GetByName()** (**GetByNameAsync()**) and **GetByKey()** (**GetByKeyAsync()**) respectively. There is also fastest way to get an instance of the current user by calling the **Current** property in the **StiServerConnection.Accounts.Users** class.
